Table Crasher

Table Crasher is a full-stack (Postgres, Ruby on Rails, React/Redux) web application inspired by OpenTable. It implements AJAX to get real time availability of reservations from the database. Table Crasher also has integrated AWS for image storage to keep local storage to a minimum, and a customized query search to find reservation availability by restaurant or city.

Data Ghost

A Ruby Database ORM framework which maps common SQL queries to Ruby methods. It also features associations to simply relate database tables. Data Ghost utilizes meta-programming to dynamically define and calls methods for custom search

  • Github

MagnaBall

Magna ball is an interactive, 1 or 2 player 2D physics game. The physics sytem is custom built to allow greater control over ball trajectory, collisions, and magnets. There is an AI system to allow for single or two player functionality. MagnaBall leverages EaselJS to draw and update graphics quickly and responsively.
